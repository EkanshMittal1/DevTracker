# DevTrack

A full-stack issue tracking application inspired by Jira and Linear.

## Features

- JWT Authentication
- User Registration & Login
- Create, Update & Delete Issues
- Issue Search & Filtering
- Protected API Routes
- MongoDB Database
- Responsive UI
- User-specific Workspaces

## Tech Stack

Frontend
- React
- Axios
- Context API

Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
